# State Population Analyzer

## Overview
A Python project that analyzes population data for different Indian states using lists, dictionaries, loops, and conditional statements.

## Features
- Finds the highest population
- Finds the lowest population
- Identifies corresponding state names
- Calculates average population

## Technologies Used
- Python

## Concepts Practiced
- Lists
- Dictionaries
- For Loops
- If Statements
- Variables
- len()
